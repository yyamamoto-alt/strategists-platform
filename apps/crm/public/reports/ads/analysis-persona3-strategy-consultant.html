Executive Summary

結論: Google広告は「投資対効果は出ている」が、事業全体での位置づけは限定的

3つのキーファインディング

  1. ユニットエコノミクスは成立している。 CAC ¥66.9万に対し、確定売上+期待紹介料ベースのLTVは¥97.4万。紹介料が実現すれば投資回収は可能。ただし紹介料確定ゼロの現状ではROAS 0.34倍で赤字。
  2. 事業全体への貢献度は2.2%。 全317成約のうちGoogle広告経由は7件。「あれば嬉しい」チャネルであって「事業の柱」ではない。他チャネル（紹介・SEO等）が圧倒的に主力。
  3. 9月以降のCV急増はデータの信頼性問題。 CPA ¥1,400まで改善したように見えるが、マイクロCVの混入が原因の可能性が高い。実質CVは月2-3件ペースで変わっていない。

So What?（だから何なのか）

  • Google広告は現状の月4万円予算なら「低リスクの実験枠」として許容範囲
  • ただし月100万円に戻す判断は、紹介料の実現可能性が見えてからで十分
  • 事業成長のレバーはGoogle広告ではなく、既存チャネルの最適化にある

Now What?（次に何をすべきか）

  • 即時: コンバージョン計測の正常化（マイクロCVを分離）
  • 短期: 紹介料パイプライン¥521万の実現状況を追跡
  • 中期: 月40万円規模で「ケース面接 対策 講座」系KWに集中投資

構造的課題

紹介料確定ゼロの意味: 34件のエージェントレコードがあるが、確定件数ゼロ。期待紹介料¥521万は「絵に描いた餅」の状態。紹介料ビジネスの転換率をまず把握すべき。
CAC計算の補正: 広告費¥468万 / CRM顧客34名 = 顧客獲得単価¥13.8万。しかし成約ベースでは ¥468万 / 7成約 = ¥66.9万/成約。ROAS目標3.0xの場合、成約あたりLTV ¥200万が必要だが、現在の平均成約単価は¥22.9万。約9倍のギャップがある。

Google広告の「役割」はどこにあるか？

仮説A: 量のチャネル

大量リードを安く獲得し、その中から成約を絞り込む。

→ 否定。年間34名では量とは言えない。全1,157名の3%。

仮説B: 質のチャネル

他では取れない高LTV顧客を獲得する。

→ 否定。成約率・実施不可率とも全体平均を下回る。

現実的な位置づけ: Google広告は「ブランド認知が届かない層（ケース面接対策を能動的に検索する層）」への接点。量も質も主力には及ばないが、オーガニックでは取りこぼす"能動検索層"へのリーチとしては一定の意味がある。月4万円規模なら維持する合理性あり。
Analysis 03

月次トレンド: 2025年9月に「構造変化」が起きたのか？

9月以降のCPA急落は、実力改善ではなくコンバージョン定義の変更が主因と推定される。

月次データの「見え方」と「実態」

データの整合性に重大な疑念:
Phase 2以降、クリック数が1/4以下に減少したにも関わらず、CVが3倍に増加している。これは「btn_click」「マイクロCV」の計測追加によるCV水増しの可能性が極めて高い。

根拠:
  • コンバージョンアクション内訳: btn_click 164件 + マイクロCV 117件 = 281件（全CVの72%）
  • 実質的な申込CV（既卒遷移39件 + 新卒遷移18件 + thanks13件）= 70件が「本物」
  • 9月以降のCV急増タイミングとbtn_click導入時期が一致

Analysis 07

投資判断: 3段階のアクションプラン

まず「測定を正す」、次に「効率を上げる」、最後に「規模を拡大する」。順序を間違えてはならない。

1
即時（今週中）

計測基盤の正常化

  • マイクロCV・btn_clickをコンバージョン列から除外（レポート用に残すのは可だが、入札最適化には使わない）
  • 「既卒遷移」「新卒遷移」「thanks遷移」のみを主CVとして設定
  • CRMとの月次突合プロセスを確立（Google Ads CV vs CRM流入数の差異管理）

これをやらないと、以降の全ての最適化判断が歪む。

2
短期（1ヶ月以内）

投資効率の最大化

  • CV=0キーワードの即時停止/除外: 「コンサル転職エージェント」「BCG転職」等 → 年間¥25万の節約
  • 高効率KWへの集中: 「戦略コンサル 対策」「ケース面接 講座」系
  • 曜日入札調整: 木・水・金に+20%、月・火に-20%
  • デスクトップ入札調整: +15%
  • 予算を月20万円に引き上げ（Phase 1完了後）
3
中期（3ヶ月後）

スケール判断

  • Phase 2の実質CPA（CRM突合ベース）が¥10万以下なら、月40万円に増額
  • 紹介料の実現率が50%以上なら、月100万円への復帰を検討
  • 判断基準: 確定売上+実現紹介料ベースのROAS 1.5x以上

データが揃わないうちの増額は「賭け」であり「投資」ではない。

経営者への最終メッセージ

Google広告に期待しすぎてはいけない

現状、事業の97.8%は広告以外のチャネルで成り立っている。Google広告は「補完チャネル」であり、ここに事業の命運を賭けるべきではない。主力チャネル（紹介・SEO等）の強化が最優先。

ただし、捨てる必要もない

月20万円規模で効率的に運用すれば、年間数十名の「能動検索層」にリーチできる。紹介料ビジネスが軌道に乗れば、Google広告経由の顧客もLTV ¥97万以上になり得る。紹介料の実現がこのチャネルの命運を握る。

Appendix

補足: 主要な前提と留意事項

データの制約

  • Google Ads上のCV数（177件）とCRM上の顧客数（34名）には大きな乖離がある。本レポートでは事業判断にはCRMデータを優先的に使用。
  • 紹介料¥521万は「期待値」であり確定ゼロ。紹介料ビジネスのパイプライン管理データが不足しているため、実現確率の推定は行っていない。
  • 2025年9月前後でコンバージョン計測定義が変更された可能性が高く、月次トレンドの前後比較は単純にはできない。
  • Google広告以外のチャネル別データがないため、チャネル間の比較は全体平均との対比にとどまる。
