# 顧客データベースシステム 要件定義書

## 1. システム概要

戦略コンサル転職塾・エージェントサービスの顧客管理を一元化するWebシステム。
既存のGoogleスプレッドシートによる管理を置き換え、営業パイプライン・学習管理・転職支援・売上管理を統合的に扱う。

## 2. 技術スタック

| 項目 | 技術 |
|---|---|
| フロントエンド | Next.js 14 (App Router) + TypeScript |
| UI | Tailwind CSS + shadcn/ui |
| データベース | Supabase (PostgreSQL) |
| 認証 | Supabase Auth |
| ホスティング | Vercel (予定) |
| チャート | Recharts |

## 3. 事業セグメント

1. **既卒スクール×エージェント事業** (メイン)
2. **新卒スクール事業**
3. **MyVision業務委託**
4. **コンテンツ販売**

## 4. 営業パイプライン

```
問い合わせ(申込) → 日程確定 → 初回面談実施 → 提案/追加指導 → 契約(成約) → 入金
```

## 5. 料金体系

- 入会時一括の受講料（分割払いあり）
- 追加オプション
- 補助金対応（経産省）
- 人材紹介成約時のボーナス（紹介手数料）

## 6. ユーザー権限

初期フェーズは全員同じ権限（将来的にロール分離可能な設計）

## 7. 機能一覧

### Phase 1: 営業・CRM（優先）
- [ ] 顧客一覧（検索・フィルタ・ソート）
- [ ] 顧客詳細（カルテ画面）
- [ ] 営業パイプライン（カンバン表示）
- [ ] 営業活動記録（面談記録・メモ）
- [ ] KPIダッシュボード（ファネル分析）

### Phase 2: 契約・売上管理
- [ ] 契約管理（プラン・金額・割引・分割）
- [ ] 入金管理・請求状況
- [ ] 売上ダッシュボード（P/L）
- [ ] 補助金管理

### Phase 3: 学習管理（LMS統合）
- [ ] 指導スケジュール管理
- [ ] カリキュラム進捗追跡
- [ ] 出席率・評価管理
- [ ] レベル判定

### Phase 4: エージェント・転職支援
- [ ] 転職活動状況管理
- [ ] 企業応募・選考管理
- [ ] 内定・入社管理
- [ ] 人材紹介手数料管理

## 8. KPI指標（P/Lシートから）

### ファネルKPI
- 申込数（チャネル別: SEO, Blog, X, Youtube, note, ココナラ, 紹介, 広告等）
- 日程確定数 / 日程確定率
- 面談実施数 / 実施率
- 成約数 / 成約率

### 収益KPI
- 売上（確定/見込）
- LTV（スクール確定分 / 人材見込含む）
- 客単価
- 人材紹介併用率
- CPA（チャネル別）

### 学習KPI
- 月間指導回数
- 出席率
- レベルアップ率

## 9. 顧客DBカラムマッピング（既存スプレッドシート → 新システム）

### 基本情報 (A〜H)
- 申込日、名前、メール、電話番号、utm_source、utm_medium、utm_campaign、属性(既卒/新卒)

### 営業情報 (I〜W)
- 経歴、申込時点エージェント希望、面談予定、商談状況、終了/未確定/確定
- 実施状況、初回面談結果、申し込みの決め手、営業日、成約、入金日、営業内容

### CRM詳細 (W〜AF)
- 営業内容、営業方針、90日メッセージ、エージェント確認
- 経路(営業担当記入)、比較サービス、二次商談分類、延期期日、入会時期/リードタイム

### 契約情報 (AG〜AQ)
- 一次/二次/確定金額、割引、Progress Sheet
- 受講プラン、変更プラン、入金日、指導終了日

### 学習管理 (AR〜BD)
- 指導開始日、指導終了日、最終指導日、累計指導回数
- アクセスメント、出席率、最新評価、レベル(初級者/中級者/上級者)、選考状況

### 転職支援 (BE〜BW)
- レベルアップ確認、書類合格率、受験数、PV定数
- 最終結果、最終初期レベル、入金フォーム、確定金額
- エージェント業務メモ、就活転職活動結果・内定先

### エージェント詳細 (BX〜CE)
- 利用エージェント、入社先就職先、内定年収、想定年収率
- 紹介手数料率、マージン、入社予定日、メモ

### プロフィール詳細 (CF〜CZ)
- プリオリティ、志望企業一覧、申込時レベル
- ケース面接対策の進捗、申し込みの決め手(カフ)
- SNSアカウント、参考メディア、趣味・特技、行動特性、備考、注意事項

### 学習詳細指標 (DA〜DR)
- 入会月、受講料、販売金額、契約金額
- 指導回数、受験数、年齢
- エージェント失注理由、詳細、請求状況
- 初回面でのE募の有無、データベース確定日

### 運用管理 (DS〜EK)
- 管理者確認、書類提出状況・日、転職紹介先確保
- 転職先確保日、大学名、学部、メール送付内容
- 面談予定日案内、エージェント面談者、入金フォーム
- スケジュール送付日、補助金、転職紹介、人材紹介補助金付帯
